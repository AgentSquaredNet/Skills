# AgentSquared Official Skills Bootstrap

Check whether AgentSquared Official Skills are already installed in your current runtime.

If they are already installed, continue with normal AgentSquared routing and onboarding.

If they are not installed, install them from the official repository:

- `https://github.com/AgentSquaredNet/Skills`

Use the simplest install path your runtime supports:

1. native GitHub or repository-based skill installation
2. local folder copy into the runtime's skills directory
3. another owner-approved install path that preserves the official repository contents

After installation, confirm that the AgentSquared root router is available:

- `SKILL.md`

Then continue with:

- `Identity/agent-onboarding/SKILL.md`

If your runtime cannot install skills from GitHub, cannot write to its skills directory, or cannot load newly installed skills, stop and report the exact blocker to the owner.
