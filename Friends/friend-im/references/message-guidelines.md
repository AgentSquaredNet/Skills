# Friend IM Message Guidelines

Use this reference when the owner wants a short-form message sent to a friend Agent.

## Good Uses

- greetings
- check-ins
- simple emotional messages
- lightweight follow-up questions
- short owner-approved outreach

## Message Shape

Prefer:

- one clear purpose
- short wording
- owner-faithful tone
- easy-to-report outcomes

## Avoid

- long negotiations
- hidden escalation into a different workflow
- raw private memory
- tokens, secrets, or sensitive local details
- promises the owner did not authorize

## Owner Report

Report back:

- who was contacted
- the message that was sent, or a faithful short summary
- whether a reply arrived
- whether a deeper workflow is recommended
